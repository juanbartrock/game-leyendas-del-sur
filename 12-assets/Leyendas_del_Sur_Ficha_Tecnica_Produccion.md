## 1. Resumen del producto

Juego de cartas con mazo dividido en tres tipos. Estética Urban Noir con identidad cultural argentina. Incluye reglamento, stickers, póster, infografía y figura.

## 3. Especificaciones de impresión

- **Sistema**: Offset (recomendado para tiradas >500) / Digital (para tiradas chicas o prototipos)
- **Color**: CMYK 4/4 (ambas caras a todo color)
- **Troquel**: 3 troqueles diferentes (Tarot 70×120, Póker 63×88, Mini cuadrado)
- **Diseños únicos**: 6 Forasteros + 45 Acción + 18 Dones = 69 ilustraciones frontales + 3 dorsos diferentes
- **Resolución**: Archivos PDF/X-1a a 300 dpi

## 5. Estructura completa de costos

### ✅ Lo que ya identificaste — Costos directos

| Rubro | Tipo de costo | Notas |
|---|---|---|
| **Impresión de cartas** (3 mazos) | Unitario | 3 tamaños diferentes, 3 troqueles |
| **Caja 2 piezas** impresa | Unitario | Incluye forrado |
| **Reglamento** | Unitario | Impresión + plegado/encuadernado |
| **Stickers** | Unitario | Impresión + troquel |
| **Póster** | Unitario | Impresión plegada |
| **Infografía** | Unitario | Impresión |
| **Figura** | Unitario | **Este puede ser el item más caro por unidad** |
| **Flete** | Logística | Marítimo China → Argentina, puerta a puerta |
| **Diseño** | Honorarios | Ilustraciones, layout de cartas, diagramación reglamento |
| **Publicidad** | Marketing | Ads Meta/Google, Adfound |
| **Comisiones** | Plataforma | Gamefound: ~8% (5% plataforma + 3% procesamiento) |

### ❌ Lo que falta considerar

| Rubro | Tipo | Por qué importa |
|---|---|---|
| **🔧 Setup / herramientas** | One-time | Troqueles para cada tamaño de carta ($100-$300 c/u). Planchas de impresión ($200-$1.000). Se paga una vez pero hay que presupuestarlo |
| **📦 Embalaje para distribución** | Unitario | Cajas de cartón corrugado para enviar a cada backer. Burbuja / protección. Cinta. No es lo mismo que la caja del juego |
| **🛃 Arancel de importación + IVA** | % del valor | **ENORME en Argentina**: DAI (~35%) + IVA (21%) + anticipo Ganancias + IIBB. Sobre el valor CIF (producto + flete). Puede sumar **40-55% adicional** |
| **🧾 Gestoría aduanera / despachante** | Honorarios | Alguien tiene que gestionar la importación en aduana argentina (~$300-$800) |
| **🚚 Envío interno Argentina** | Por backer | Costo de enviar cada juego desde el depósito a la casa del backer (Correo Argentino, Andreani, etc.). Puede ser $5-$15 USD por envío |
| **🎯 Prototipado / muestras** | One-time | Mandar a hacer samples antes de la tirada final para verificar calidad de impresión, colores, troquel. ~$200-$500 |
| **🔁 Merma / reposiciones** | % del total | Siempre hay cartas defectuosas, cajas rotas en flete. Calcular 3-5% extra de reposición |
| **💵 Contingencia cambiaria** | Buffer | Si fabricás en USD y vendés en ARS, el tipo de cambio puede moverse. Un 10-15% de buffer no es exagerado |
| **🎬 Producción de campaña** | One-time | Video promocional, fotografía de producto, diseño de la página de Gamefound |
