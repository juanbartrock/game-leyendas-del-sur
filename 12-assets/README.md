# 12 — Assets

> Archivos multimedia: imágenes, capturas, referencias visuales, sonidos.

## [[../HOME|← Volver al índice]]

---

Esta carpeta contiene o referencia imágenes que usamos en las notas del vault. Como Obsidian puede mostrar imágenes locales, podemos guardar acá:

- Capturas de pantalla del proyecto en Godot
- Outputs de ComfyUI (personajes, escenarios, concept art)
- Moodboards y referencias
- Diagramas
- Screenshots de tutoriales

> ⚠️ **Ojo con el tamaño:** los .png pesados pueden hinchar el repo. Preferí .jpg para fotos/referencias y .webp para assets editados. Si se vuelve muy grande, evaluamos usar una carpeta externa con symlinks.
