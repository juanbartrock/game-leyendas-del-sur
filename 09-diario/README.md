# 09 — Diario de Desarrollo

> Notas cronológicas de cada sesión. Progreso, decisiones, reflexiones, frustraciones, descubrimientos.

## [[../HOME|← Volver al índice]]

---

Cada sesión de trabajo queda registrada en una nota con fecha `YYYY-MM-DD.md`. El formato es libre, pero idealmente incluye:

- ¿Qué hice hoy?
- ¿Qué aprendí?
- ¿Qué me frenó / trabó?
- ¿Qué decidí?
- ¿Qué sigue?
